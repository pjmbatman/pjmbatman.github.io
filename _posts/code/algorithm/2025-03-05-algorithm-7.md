### 문제 풀이

- 정렬을 통해 우선순위나 조건을 빠르게 파악하기 위한 접근이기 때문입니다.

```python
# n편중 h편이상이 h번 이상 인용
# n-h 편이 h이하 인용
# 내림차순으로 정리하고 인덱스+1 == arr[인덱스] 를 비교
def solution(citations):
    
    citations.sort(reverse=True)
    
    for i, citation in enumerate(citations):
        if i+1 <= citation:
            continue
        else:
            return i
        
    return len(citations)
```
